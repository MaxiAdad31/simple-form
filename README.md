# Cuestionario del Puestero

Este es un cuestionario interactivo diseñado para capturar la experiencia y conocimientos de los puesteros del Mercado de Abasto de Córdoba. El formulario está estructurado en 6 secciones con 29 preguntas que cubren aspectos del negocio actual, clima, clientes, momentos clave, colegas y visión futura.

## Características

- **Interfaz responsiva**: Funciona en dispositivos móviles y de escritorio.
- **Modo oscuro**: Se adapta automáticamente al tema del sistema operativo.
- **Progreso en tiempo real**: Muestra el porcentaje de completitud del cuestionario.
- **Resumen interactivo**: Genera un resumen de las respuestas al finalizar.
- **Impresión optimizada**: Diseñado para imprimir en PDF.

## Archivos

- `index.html`: Archivo principal del cuestionario.
- `css/styles.css`: Estilos CSS separados para mejor mantenibilidad.
- `js/script.js`: Lógica JavaScript para la interactividad.
- `README.md`: Documentación del proyecto.

## Uso

1. Abre `index.html` en un navegador web.
2. Completa las secciones expandiendo cada bloque.
3. Haz clic en "Finalizar y ver resumen" para generar el resumen.
4. Usa "Imprimir / PDF" para guardar una copia.

## Desarrollo

El proyecto utiliza HTML5, CSS3 y JavaScript vanilla. Los estilos están separados en `styles.css` y la lógica en `script.js` para facilitar el mantenimiento.
